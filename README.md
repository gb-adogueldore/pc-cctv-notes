# pc-cctv-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容与 PC 设备、监控系统、笔记整理等相关，不针对任何特定域名或网站。

## 项目简介

本仓库的主要目的是以静态 HTML 页面的形式，保存和展示一系列独立的文档或工具页面。每个页面均为完整的、自包含的 HTML 文件，可直接在浏览器中打开查看。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 主入口页面（可选）
├── pages/             # 存放归档的 HTML 页面文件
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 存放页面所需的静态资源（图片、样式、脚本等）
    ├── images/
    ├── styles/
    └── scripts/
```

- `pages/` 目录用于存放归档的独立 HTML 页面，每个页面均独立完整。
- `assets/` 目录用于存放页面引用的静态资源，按类型分文件夹组织。

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，不依赖于后端服务或外部数据库。
- 页面之间可能存在链接关系，但每个页面均可单独打开使用。
- 归档的页面内容以技术笔记、工具页面、说明文档为主，不包含任何商业推广或诱导内容。
- 页面内不包含任何具体域名、网址、关键词或营销承诺。

## 维护说明

- 本仓库由个人维护，更新频率不固定。
- 欢迎提交 Issue 或 Pull Request 来修正错误或补充内容。
- 由于页面为静态 HTML，修改后直接提交即可，无需额外构建步骤。
- 请勿在页面中添加任何形式的广告、跟踪代码或恶意内容。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/pc-cctv-notes.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
